A newfound knowledge

Star Wars is one of the most popular Sci-Fi franchises in the world with a vast universe of planets, jedi, alien species, and anything else you can think of.

There is always something interesting and new to learn about within the Star Wars universe. This web app was built to help with diving into anything you are curious about within this universe.

To begin, select if you would like to search for a planet, character, or ship. You can also generate random info cards if you want to start learning but don't have anything specific in mind

This app was developed by Devon McDonald using AngularJS and the swapi.
